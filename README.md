# Support Ticket Classification - Machine Learning Project

This project builds a machine learning model to classify customer support tickets into categories.

## Categories
- Billing
- Account
- Technical

## Tools Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Workflow
1. Data loading
2. Text preprocessing
3. Feature extraction using TF-IDF
4. Model training
5. Prediction and evaluation

## Example Prediction
Input: Payment deducted but order not placed  
Output: Billing
